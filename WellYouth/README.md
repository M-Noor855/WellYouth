# WellYouth - Wellness for the Next Generation

WellYouth is a modern, holistic wellness platform designed to help young people manage their mental and physical health through habit tracking, mindful activities, and community support.

## 🌟 Key Features

- **Personalized Habit Tracking:** Set daily goals for hydration, sleep, and movement.
- **Interactive Wellness Activities:**
  - **Calm & Regulate:** Guided breathing timers and mindfulness minutes.
  - **Daily Essentials:** Hydration challenges and quick stretch routines.
  - **Positive Mindset:** Gratitude journaling and intentional focus exercises.
- **AI-Powered Assistant:** A dedicated floating assistant to provide guidance and support.
- **Community Support:** Join groups and engage with peers in a safe, moderated environment.
- **Specialist Access:** Connect with wellness professionals when needed.
- **Gamified Progress:** Earn points and level up by completing activities and maintaining streaks.

## 🛠️ Technical Stack

- **Framework:** ASP.NET Core 10.0 (MVC)
- **Database:** Entity Framework Core with SQL Server
- **Frontend:** Vanilla CSS (Modern CSS Variables), JavaScript (ES6+), Razor Views
- **Icons:** FontAwesome 6

## 🚀 Getting Started

### Prerequisites
- [.NET 10 SDK](https://dotnet.microsoft.com/download)
- [Visual Studio 2022](https://visualstudio.microsoft.com/) (recommended) or VS Code

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YourUsername/WellYouth.git
   ```
2. Navigate to the project folder:
   ```bash
   cd WellYouth
   ```
3. Update the database:
   ```bash
   dotnet ef database update
   ```
4. Run the application:
   ```bash
   dotnet run
   ```

## 📂 Project Structure
- `Controllers/`: Application logic and request handling.
- `Models/`: Data entities and view models.
- `Views/`: Razor templates for the user interface.
- `Services/`: Business logic layer (Scoring, Recommendations, etc.).
- `wwwroot/`: Static assets (CSS, JS, Images).
